# Installation
`
git clone https://github.com/im-snowin/react-page-transition.git my-app
`
# Installing dependencies
Using npm
`
npm install
`
Using yarn
`
yarn
`
`
cd my-app
`

# Start the development env
`
npm run dev
`
or
`
yarn dev
`
